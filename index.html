HTML

HyperText is just a fancy word for text that links to something else.
Markup Language is what marks up the text with the instructions written inside of these angle brackets <>, known as tags.
The actual language of HTML is the instructions that we wrap around the text to format it and as we'll see, instructions for many other things like images, links, videos and more.
